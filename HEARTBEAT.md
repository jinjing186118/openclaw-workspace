# HEARTBEAT.md - Daily Checklist

## 每晚任务 (23:00)

- [ ] 读取今天的记忆文件 `memory/YYYY-MM-DD.md`
- [ ] 整理当日重要事件、决策、学习点
- [ ] 如有需要，更新 MEMORY.md 长期记忆
- [ ] 记录心跳执行状态到 `memory/heartbeat-state.json`

## 工作日志提醒 (17:30)

- [ ] 执行 `python3 scripts/workday_reminder.py`
- [ ] 脚本会自动判断今天是否是中国工作日（考虑节假日和倒休）
- [ ] 如果是工作日，生成日报并发送提醒给金主人

## 工作日志记录方式

金主人可以随时通过以下方式记录工作：

### 方式一：直接跟我说
随时随地发消息描述今天做了什么，例如：
- "上午跟研发开需求评审会，2.5小时"
- "下午调研OpenClaw，试了定时任务"
- "写了技术方案文档"

我会自动扩写成专业的工作日志格式。

### 方式二：使用命令记录
```bash
python3 scripts/log_work.py "工作描述"
```

### 日报生成
- 17:30 自动生成的日报会包含已记录的所有工作
- 工作要点已自动包装和扩写
- 直接复制到易云表格即可提交

## 执行规则

1. **轻量整理** - 不要过度处理，保留原始感受
2. **关注重点** - 金主人的情绪、重要决策、新技能/配置
3. **定期归档** - 每周回顾一次，将精华写入 MEMORY.md
4. **保持简洁** - 心跳任务尽量轻量，避免 token 浪费
