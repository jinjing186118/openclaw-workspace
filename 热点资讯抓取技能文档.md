## 五、微信公众号抓取（核心技术）

### 自动降级流程

```
Step 1: web_fetch尝试抓取
Step 2: 返回内容<200字 → 自动降级Playwright
Step 3: Playwright + 移动端UA抓取
Step 4: 返回完整内容
```

### Playwright抓取代码

```javascript
const { chromium } = require('playwright');
(async () => {
  const browser = await chromium.launch({
    headless: true,
    args: ['--no-sandbox', '--disable-setuid-sandbox']
  });
  const ctx = await browser.newContext({
    userAgent: 'Mozilla/5.0 (iPhone; CPU iPhone OS 16_0 like Mac OS X) AppleWebKit/605.1.15 (KHTML, like Gecko) Version/16.0 Mobile/15E148 Safari/604.1'
  });
  const page = await ctx.newPage();
  await page.goto(URL, {
    waitUntil: 'domcontentloaded',
    timeout: 20000
  });
  await page.waitForTimeout(3000);
  const content = await page.evaluate(() => {
    const sels = ['#js_content', '.rich_media_content', '#js_article', 'article'];
    for (const s of sels) {
      const e = document.querySelector(s);
      if (e && e.innerText.length > 100) return e.innerText;
    }
    return document.body.innerText;
  });
  console.log(content);
  await browser.close();
})();
```

### 关键点
- 必须用移动端UA（iPhone），桌面UA触发验证码
- waitUntil用domcontentloaded，不用networkidle
- 额外等3秒让JS渲染正文
- 正文在#js_content元素中
- 环境：Playwright 1.58.2已安装

### 失败方案记录（避坑）

| 方法 | 结果 | 原因 |
|------|------|------|
| web_fetch | 只拿到标题 | JS渲染 |
| Playwright+桌面UA | 验证码拦截 | 微信检测非移动端 |
| Google缓存 | 无缓存 | 微信文章不被缓存 |
| 搜狗微信搜索 | 非原文 | 不返回完整正文 |
| 百度搜索 | 仅摘要 | 可作备选拼凑 |
